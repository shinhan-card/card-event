## ⚠️ 참고사항

### API 키 관련
- 제공하신 Gemini API 키가 만료되었습니다
- 새로운 API 키 발급이 필요합니다: https://ai.google.dev/
- `.env` 파일에서 `GEMINI_API_KEY` 업데이트 필요

### 실제 크롤링 활성화 방법
1. 각 카드사 사이트 구조 분석
2. `scraper.py`의 CSS 셀렉터 조정
3. `quick_test.py`로 1개 카드사 테스트
4. 성공 시 `main.py --once` 실행
